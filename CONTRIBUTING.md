# Contributing

Contributions welcome. This toolkit aims to be the most practical open-source resource for EU AI Act compliance.

## What we need

- Corrections to legal interpretations (cite the relevant Article)
- Additional worked examples for common AI systems
- Translations (French, German, Spanish, Italian are priorities)
- Integration guides for existing tools (AIF360, Fairlearn, etc.)
- Harmonised standards mapping as CEN/CENELEC JTC 21 standards are published

## How to contribute

1. Fork the repo
2. Make your changes
3. Submit a PR with a clear description

## Style

- Plain language. Avoid legal jargon where possible.
- Actionable. Every section should tell the reader what to DO.
- Cite Articles. Reference specific EU AI Act articles for every requirement.
- No filler. If a sentence doesn't help someone achieve compliance, cut it.
