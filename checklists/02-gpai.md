# General-Purpose AI Model Compliance Checklist

Your model is classified as a General-Purpose AI model under the EU AI Act (Article 51-53).

**Enforcement date: August 2, 2025**

---

## All GPAI Providers Must:

- [ ] Prepare and maintain technical documentation of the model (Annex XI)
  - [ ] Training methodology and techniques
  - [ ] Design choices and assumptions
  - [ ] Training data sources (general description, including key design choices and data provenance)
  - [ ] Computational resources used for training
- [ ] Prepare and maintain information and documentation for downstream providers integrating the model
- [ ] Comply with EU copyright law (Directive 2019/790), including text and data mining opt-out provisions
- [ ] Publish a sufficiently detailed summary of training data content (template from AI Office)

## Open-Source Exception

If you make your model available under an open-source license (parameters, architecture, usage info publicly available):

- [ ] Only copyright compliance + training data summary required
- [ ] Technical documentation requirements are waived
- [ ] UNLESS your model is designated as posing systemic risk (then full obligations apply)

## Code of Practice

- [ ] Consider adhering to the Code of Practice for GPAI providers (being developed through AI Office-facilitated process, expected late 2025)
